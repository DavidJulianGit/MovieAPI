Documentation

Developed by: David Schindler

January 2024

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

API Endpoint Documentation

Operation Request URL Response
Return a list of all movies GET /movies JSON array of all movies with their details
Return all data about a single movie by title GET /movies/:title JSON object containing details of the specified movie
Return description of a genre by name GET /genres/:name JSON object containing description of the specified genre
Return data about a director by name GET /directors/:name JSON object containing bio, birth year and death year of the specified director
Allow new users to register POST /register JSON object confirming a successfull registration
Allow users to update their user info PUT /users/:userId JSON object confirming the successfull update
Allow users to add a movie to favorites PUT /users/:userId/favorites Text confirmation of the movie being added to favorites
Allow users to remove a movie from favorites DELETE /users/:userId/favorites/:movieId Text confirmation of the movie being removed from favorites
Allow existing users to deregister DELETE /users/:userId Text confirmation of the user being deregistered
